==========================================
TENDER DOCUMENT GENERATOR
==========================================


СОДЕРЖИМОЕ ПАПКИ:
=================
TenderDocGenerator/
├── TenderDocGenerator.exe    ← ЗАПУСКАЙТЕ ЭТОТ ФАЙЛ
├── _internal/                 ← Системные файлы Python
├── data/                      ← Примеры JSON для тестирования
│   ├── profile.json           (данные компании)
│   ├── tender.json            (данные закупки)
│   └── calc.json              (расчёты цен)
├── templates/                 ← Шаблоны документов
│   ├── 01_Анкета_участника_шаблон.docx
│   ├── 02_Заявка_на_участие_в_закупке_шаблон.docx
│   └── 03_Предложение_о_цене_договора_шаблон.docx
└── output/                    ← Одна из папок куда выбрать сохранить результаты

КАК ИСПОЛЬЗОВАТЬ:
=================
1. Распакуйте архив TenderDocGenerator.zip
2. Откройте папку TenderDocGenerator
3. Запустите файл TenderDocGenerator.exe
4. В открывшемся окне:
   - Шаг 1: Выберите 3 JSON-файла с данными (папка data)
   - Шаг 2: Выберите папку с шаблонами DOCX (папка template)
   - Шаг 3: Выберите папку для сохранения (создана папка output)
   - Нажмите зелёную кнопку "СГЕНЕРИРОВАТЬ ДОКУМЕНТЫ"
5. Проверьте сгенерированные документы в папке output или директории в которую указали
